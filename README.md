# so_long 🕹️

A 2D game project for the 42 Cursus, developed using **C** and the **MiniLibX** library.

## 🕹️ Mandatory Part
The goal is to create a small 2D game where the player collects all items and reaches the exit.

![Mandatory Gameplay](![alt text](<Screen Shot 2026-05-03 at 1.34.47 PM.png>))

## 🌟 Bonus Part
The bonus version includes extra features like:
*   **Patrol Enemies**: Moving obstacles that end the game on contact.
*   **On-screen HUD**: Displaying the move count directly in the window.
*   **Animations**: Animated sprites for characters and collectibles.
